Testing Sinatra with RSpec
==========================

Requiremnts for this are Sinatra, Rspec and rack-test

    gem install sinatra rspec rack-test

from here you can run:

    ruby app.rb

Then visit 127.0.0.1:4567

To run the test suite
---------------------

    cd ./spec
    spec myapp_spec.rb
